# Bootstrap
## Build a company. Start with a product. Ship it. 

A HackClub YSWS, By Johnathan Mcphilbin

live site: https://bootstrap-production-e275.up.railway.app

## What is it?
Bootstrap is a YSWS where teenagers build and ship a real product with the intent to make money - a SaaA, Game, CLI tool, Chromeextension etc
Doesnt have to be profitable it is about the idea and the ability to follow through 

## Rules:
Live URL + open source code (Use Railway for free hosting for submission)

Open Source code

At least 5 real users who aren't you

A 2-minute pitch explaining what you built, who it's for, and how it makes money

Time is tracked (Hackatime or Lapse)

## Scoring (out of 100)

| Category | Points |
|---|---|
| Live URL + open source code | 20 |
| 5 real users (not you) | 25 |
| 2-min pitch quality | 25 |
| Revenue model | 20 |
| Hackatime hours | 10 |
