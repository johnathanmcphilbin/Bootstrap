# Bootstrap
## Build something real. Ship it. Get paid.

A Hack Club YSWS by Johnathan McPhilbin

live site: https://bootstrap-production-e275.up.railway.app

## What is it

Bootstrap is a Hack Club YSWS where teenagers build and ship a real product with the intent to make money. SaaS, game, CLI tool, Chrome extension, hardware — anything goes as long as it has a live URL and real users. It does not have to be profitable. It has to be real.

## Rules

- Live URL and open source repo
- At least 5 real users who are not you
- A 2-minute pitch video explaining what you built, who it is for, and how it makes money
- Time tracked with Hackatime or Lapse
- Must be under 18
